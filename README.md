# Personal Finance Manager

A Python-based personal finance manager that helps users track their income and expenses, categorize them, and visualize spending trends. The tool stores transactions in a SQLite database and provides financial summaries and visualizations.

## Features:
- Add income and expenses with categories and descriptions.
- View all transactions.
- Categorize expenses and visualize spending patterns.
- Generate financial summaries showing income, expenses, and net savings.
- Visualize expenses by category using graphs.

## Technologies Used:
- Python
- SQLite (for database storage)
- Pandas (for data manipulation)
- Matplotlib / Seaborn (for data visualization)

## Getting Started:

### Prerequisites:
Ensure that Python 3.x is installed. You will also need to install the required dependencies.
